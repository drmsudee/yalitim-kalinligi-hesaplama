# Optimum Yalıtım Kalınlığı Hesaplama Programı

Bu proje, silindirik boru sistemlerinde optimum yalıtım kalınlığını hesaplamak amacıyla Python ve Tkinter kullanılarak geliştirilmiş bir masaüstü uygulamasıdır.

## Özellikler
- Tek katmanlı yalıtım optimizasyonu
- Çift katmanlı yalıtım optimizasyonu
- Isı kaybı hesaplama
- Yıllık enerji maliyeti analizi
- Yatırım maliyeti hesaplama
- Kullanıcı dostu grafik arayüz

## Kullanılan Teknolojiler
- Python
- Tkinter
- Math Kütüphanesi

## Programı Çalıştırma

```bash
python yalitim.py
```
